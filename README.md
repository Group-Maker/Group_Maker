## 🎲 How to Use

- Manage Members Page

  <img width="1392" alt="Manage Members Page" src="https://user-images.githubusercontent.com/70943835/204119065-2b85f509-b032-4c87-b9d0-923d4779227a.png">

  - Add the names of the members participating in group activities. Duplicate names are not allowed.
  - 팀프로젝트에 참여중인 멤버의 이름을 추가해주세요. 중복되는 이름은 허용되지 않습니다.
  
  - Double-click to change the member's name. The name of the member in the previous records also will be changed.
  - 더블클릭으로 멤버의 이름을 수정할 수 있습니다. 지난 기록에서도 해당 멤버의 이름이 수정됩니다.
  
  - You can delete a member with the X button that appears when you hover the mouse over the member's name. The member in the previous records will not be deleted. Deleted name cannot be used again, so be careful!
  - 멤버의 이름에 마우스를 호버했을 때 나타나는 X 버튼으로 멤버를 삭제할 수 있습니다. 지난 기록에서는 멤버가 삭제되지 않습니다. 이후 해당 이름은 다시 사용할 수 없으니 주의하세요!
  
- Previous Records Page

  <img width="1392" alt="Previous Records Page" src="https://user-images.githubusercontent.com/70943835/204120975-4aae9e96-d8d3-4b34-8f48-b0bf9a057485.png">
  
  - You can check and delete previous group project records. The deleted record cannot be recovered, so be careful!
  - 지난 팀프로젝트 기록을 확인하고 삭제할 수 있습니다. 기록을 삭제할 경우 다시 되돌릴 수 없으니 주의하세요!

- Make New Group Page

  <img width="1392" alt="Make New Group Page" src="https://user-images.githubusercontent.com/70943835/204121425-e43badfe-cd38-4d8d-9930-102cbed2c16a.png">

  - Determine the number of groups you want to create. The number of groups cannot be greater than the number of members.
  - 생성하고 싶은 팀의 수를 결정하세요. 팀의 수는 팀프로젝트에 참여중인 멤버의 수보다 클 수 없습니다.
  
  - Manually Create Groups
  
    <img width="1392" alt="Manually Create Groups" src="https://user-images.githubusercontent.com/70943835/204121442-3e5e3483-8bd5-4553-b21e-28a8c91183fc.png">
  
    - You can organize groups using drag an drop.
    - 드래그 & 드롭을 이용하여 직접 팀를 편성할 수 있습니다.
  
  - Create Optimized Groups
  
  <img width="1392" alt="Create Optimized Groups" src="https://user-images.githubusercontent.com/70943835/204121457-b11a657f-3b1d-4f3b-8412-ec95e0eff871.png">

    - We organize groups where <b>everyone can be with new people as much as possible</b>
    - 지난 팀프로젝트 기록을 기반으로 최대한 많은 멤버가 기존의 팀원들과 다시 만나지 않도록 새로운 팀을 편성해드립니다.
  
  

- You can use our website without signin! Your data will be sotred in local storage.
- 로그인하지 않고도 저희의 웹사이트를 이용하실 수 있습니다. 사용자의 데이터는 로컬스토리지에 저장됩니다.

- 
- 로그인하고 저희의 웹사이트를 이용한다면, 사용자의 데이터는 서버에 저장됩니다.
