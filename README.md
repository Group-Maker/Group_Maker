## Documentation

### Usage

- Member & Records
  - 명단을 관리하는 화면입니다
    - 인원 추가 / 수정 / 삭제가 가능합니다
  - 이전 기록을 확인할 수 있습니다
- Create Group - Auto
  - 조 짜기 버튼을 누르면 새로운 조를 자동 생성합니다.
  - 현재 멤버 목록과 이전 기록을 바탕으로 최대한 겹치지 않도록 조를 생성합니다.
    - Genetic algorithm 활용
- Create Group - Self
  - 새로운 조를 수동으로 생성합니다
